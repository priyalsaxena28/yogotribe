#  Random Cat Fact Generator

## Overview

This project is a simple and interactive web application that fetches and displays random cat facts using the Cat Fact API. It demonstrates the use of JavaScript's Fetch API, asynchronous programming, DOM manipulation, and responsive UI design with a modern dark-themed interface.

## Features

* Fetches random cat facts from a public API
* Modern dark-themed glassmorphism UI
* Responsive and user-friendly design
* Asynchronous data fetching using `async/await`
* Error handling for failed API requests
* Smooth fade-in animation for displayed facts

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Fetch API
* Cat Fact API

## Concepts Demonstrated

* DOM Manipulation
* Event Handling
* API Integration
* Asynchronous JavaScript (`async/await`)
* JSON Data Handling
* Error Handling
* Responsive Web Design
* CSS Animations and Transitions

## How to Run

1. Clone this repository.
2. Open the `index.html` file in your browser.
3. Click the **Get Cat Fact** button to generate a new random cat fact.

## Learning Outcomes

This project strengthened my understanding of frontend development by integrating a REST API with JavaScript, handling asynchronous operations, manipulating the DOM dynamically, and creating a visually appealing, responsive user interface using HTML and CSS.
